# 기능
+ 질문에 대한 답변 event-stream 응답
+ 채팅 이력 조회 및 상세 메시지 확인
+ 신규 채팅 및 기존 채팅 이어하기
+ (구현예정)사규/매뉴얼/범용 등 채팅 카테고리 세분화
+ (구현예정)범용의 경우 이미지 첨부 및 질의
+ (구현예정)채팅에 대한 사용자 평가(만족/불만족/불만족 사유등)
+ (구현예정)채팅 삭제, 채팅 제목 수정
+ (구현예정)로그인 기능, 프로필 확인
